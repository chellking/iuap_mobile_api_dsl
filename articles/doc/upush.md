# $upush
消息推送服务

## $upush.registerDevice()
注册消息推送服务

**语法**```$upush.registerDevice()```

**参数***无*

**实例**
```
$upush.registerDevice();//在某个时机里进行消息推送服务注册
```

## $upush.removeDevice()注销消息推送服务

**语法**```
$upush.removeDevice()
```

**参数***无*

**实例**
```
$upush.removeDevice();//在某个时机里进行消息推送服务注销,如button的onclick事件触发
```

详细的消息推送如何配置可以参考开发指导目录下《消息推送（Android）》
